# php Auth
Easy Basic PHP Authentication setup.

# Requirements

## System requirements
- Recent PHP
- Composer
- MySQL

# Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/uncle-b/.....git
    cd botje
    ```

2. **Install dependencies:**

    ```bash
    composer update
    ```

3. **Setup script**
    open the Auth/setupScript.php file in your browser.

4. **Testing**
    explore the basic functionality with the basic tempalate dialogs in the Auth/dialogs folder.
